Enhance your website with 640+ icons designed for Twitter Bootstrap. Just select an icon and display it in any widget on your WordPress site.

== Description ==

[Widget Icon](http://www.dfactory.eu/plugins/widget-icon/) enhances your website with 640+ beautiful icons from [Font Awesome](http://fortawesome.github.io/Font-Awesome/) and [Elusive](http://aristeides.com/elusive-iconfont/) font libraries designed for Twitter Bootstrap. You just select a font library, select an icon and display it in any widget on your WordPress site.

For more information, check out plugin page at [dFactory](http://www.dfactory.eu/) or plugin [support forum](http://www.dfactory.eu/support/forum/widget-icon/).

= Features include: =

* Works out-of-the-box with every widget and theme.
* Select from 640+ beautiful icons
* Select from Font Awesome and Elusive icon libraries
* Select icon position
* Set icon padding in pixels or percentage
* Select icon vertical allignment (top, bottom, middle etc.)
* Select icon size (standard, large, x2, x3, x4)
* Automatically use color from widget title or set it manually with color-picker
* .pot file for translations included

You can also use these icons anywhere in your theme.

= For Elusive icons =

Syntax: `<i class="elusive icon-adjust"></i>`

= For Font Awesome icons =

Syntax: `<i class="font-awesome icon-adjust"></i>`

== Installation ==

1. Install Widget Icon either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Widgets menu and select icons for each of your widgets.
